Our Service

Our App is for helping you organize your tasks. There are three type of tasks:

Dairly Tasks

This type for writing your all Daily Tasks you want to make.

Future Tasks

This type for writing your all Future Tasks which you are planning to achieve.

Extra Tasks

This type for writing your any type of task.

How Use This App

We will explain how you can use this application . Let's start!

  • Open the app on Your Mobile.
  • Select The type of task you want.
  • Write any number of tasks and it will be added to the list.
  • If you complete this task, click on it and it will be removed.
  • If you want to move to another type of task, exit the current task and Click another type.
  • If you want to know more about the app, click About.
